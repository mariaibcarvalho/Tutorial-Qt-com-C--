Download Qt no Linux versão 6

Se quer começar a desenvolver usando Qt e necessita de uma IDE leve que torna o desenvolvimento mais rápido e fácil, o Qt Creator é perfeito para isso. Caso seja usuário de Linux e queira começar com a nova versão do Qt Creator, esse tutorial é da versão 6.2, é apenas seguir seus passos.

Instalação dos Pré-Requisitos

Primeiramente, você deve instalar os pré-requisitos usando o apt gerenciador de pacotes, para isso abra um terminal (você pode utilizar as teclas de atalho CTR + ALT + T).

1º Rode esse comando para garantir que o seu gerenciador de pacotes esteja atualizado:

sudo apt-get update && sudo apt-get upgrade

2º Instale as ferramentas e bibliotecas necessárias para o Qt (copie e cole a linha inteira, ela é longa):

sudo apt-get -y install build-essential openssl libssl-dev libssl1.0 libgl1-mesa-dev libqt5x11extras5

Download do Instalador do Qt

Agora vamos instalar o Qt, vá até downloads no site oficial. O próprio site identifica que você está executando o Linux, cheque se a recomendação dele é Qt Online Installer for Linux e em seguida clique no botão verde de Download que automaticamente ele irá começar.

Executando o Instalador do Qt

Após baixar, ele provavelmente estará no seu diretório ~/Downloads e está nomeado parecido com qt-unified-linux-x64-versão-online.run. Porém, ainda é preciso torná-lo executável, para isso faça:

1º Acesse o seu diretório com o comando:

cd Downloads/

2º Em seguida use este comando para tornar o arquivo executável:

chmod +x qt* .run

3º Após isso você já pode executar o Qt usando o comando:

./qt* .run

E então o instalador do Qt irá te guiar por um conjunto de etapas, que em sua maioria você pode apenas clicar em “next” (avançar) ou em “agree” (concordo) para seguir adiante. Com exceção da etapa “Welcome” que pede para fazer o login em sua conta do Qt (caso não tenha será necessário criar uma) e da etapa “Installation Folder” que é recomendável escolher a opção “Qt 6.x for desktop development Qt libraries for GCC” e não mudar o nome ou o local do diretório em que o Qt será instalado.
